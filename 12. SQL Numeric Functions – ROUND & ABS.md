# 1️⃣ Why Do We Need Numeric Functions?

In real-world data:

- Numbers may have too many decimal places
- Values may require formatting for reports
- Negative values may appear where they shouldn’t

To transform numeric values in SQL, we use built-in numeric functions.

In this section, we focus on two essential functions:

• ROUND()
• ABS() (Absolute)

---

# 2️⃣ ROUND Function

## 🔹 What Does ROUND Do?

ROUND() adjusts a number to a specified number of decimal places.

Syntax:

```sql
ROUND(number, decimal_places)
```

---

# 🔍 Understanding How ROUND Works

Let’s take this number:

```
3.516
```

---

## ✅ Case 1: Round to 2 Decimal Places

```sql
SELECT ROUND(3.516, 2);
```

### Logic:

- Keep 2 digits after decimal → 3.51
- Look at 3rd digit (6)
- Since 6 ≥ 5 → round up

### Output:

```
3.52
```

---

## ✅ Case 2: Round to 1 Decimal Place

```sql
SELECT ROUND(3.516, 1);
```

### Logic:

- Keep 1 digit after decimal → 3.5
- Look at next digit (1)
- Since 1 < 5 → do NOT round up

### Output:

```
3.5
```

---

## ✅ Case 3: Round to 0 Decimal Places (Integer)

```sql
SELECT ROUND(3.516, 0);
```

### Logic:

- Remove decimals
- Check first decimal digit (5)
- Since 5 ≥ 5 → round up

### Output:

```
4
```

---

# 🔎 Static SQL Practice Example

```sql
SELECT
    3.516 AS original_value,
    ROUND(3.516, 2) AS round_two,
    ROUND(3.516, 1) AS round_one,
    ROUND(3.516, 0) AS round_zero;
```

### Output

| original_value | round_two | round_one | round_zero |
| -------------- | --------- | --------- | ---------- |
| 3.516          | 3.52      | 3.5       | 4          |

# 3️⃣ ABS Function (Absolute Value)

## 🔹 What Does ABS Do?

ABS() converts negative numbers to positive numbers.

Syntax:

```sql
ABS(number)
```

It returns the absolute value of a number.

---

## ✅ Example 1: Negative Number

```sql
SELECT ABS(-10);
```

### Output

```
10
```

---

## ✅ Example 2: Positive Number

```sql
SELECT ABS(10);
```

### Output

```
10
```

If the number is already positive → nothing changes.

---

# 🔎 Static SQL Practice Example

```sql
SELECT
    -10 AS original_value,
    ABS(-10) AS absolute_value;
```

### Output

| original_value | absolute_value |
| -------------- | -------------- |
| -10            | 10             |

---

# 4️⃣ Real-World Use Cases

## ✅ Use Case 1: Financial Reports

When displaying:

- Currency
- Sales
- Prices

You may want to limit decimals:

```sql
SELECT ROUND(sales_amount, 2)
FROM orders;
```

---

## ✅ Use Case 2: Cleaning Negative Values

Sometimes bad data appears:

Example:

| order_id | sales |
| -------- | ----- |
| 1        | -200  |

Negative sales may be incorrect.

You can fix it:

```sql
SELECT ABS(sales) FROM orders;
```

---

# 5️⃣ Important Notes

✔ ROUND follows standard rounding rules (5 or greater rounds up)
✔ ABS only affects negative numbers
✔ ROUND can be used on columns or static values
✔ These functions can be nested with others

Example:

```sql
SELECT ROUND(ABS(-3.516), 2);
```

Execution order:

1. ABS(-3.516) → 3.516
2. ROUND(3.516, 2) → 3.52

---

# 🎯 Summary

| Function | Purpose                   | Example        | Result |
| -------- | ------------------------- | -------------- | ------ |
| ROUND    | Control decimal precision | ROUND(3.516,2) | 3.52   |
| ABS      | Remove negative sign      | ABS(-10)       | 10     |

---

# Core Concept

ROUND → Formatting precision
ABS → Remove negative sign

These two simple numeric functions are essential tools for data cleaning, reporting, and analytics.
