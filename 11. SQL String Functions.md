# SQL String Functions – Complete Structured Guide

This note explains how to transform and manipulate string (text) data using SQL string functions.

---

# 1️⃣ Why Do We Need String Functions?

In real databases, string values often:

- Contain unwanted spaces
- Have inconsistent letter casing
- Need formatting changes
- Need extraction of specific parts
- Require combination of multiple columns

String functions allow us to:

✔ Clean text data
✔ Standardize formatting
✔ Extract useful parts
✔ Prepare data for reporting

---

# 2️⃣ Categories of String Functions

We can divide string functions into three main groups:

---

## 🔹 A. Data Manipulation Functions

Used to modify entire string values.

Functions:

- CONCAT()
- UPPER()
- LOWER()
- TRIM()
- REPLACE()

---

## 🔹 B. Calculation Function

Used to calculate something about the string.

Function:

- LEN() (Length)

---

## 🔹 C. Extraction Functions

Used to extract part of a string.

Functions:

- LEFT()
- RIGHT()
- SUBSTRING()

---

# 3️⃣ CONCAT() – Combine Multiple Strings

## Purpose

Combines multiple string values into one single value.

### Example

```sql
SELECT CONCAT('Michael', ' ', 'Scott');
```

Output:

```
Michael Scott
```

---

### Example with Table

Task: Show first name together with country in one column.

```sql
SELECT CONCAT(first_name, ' ', country) AS name_country
FROM customers;
```

Result:

| name_country  |
| ------------- |
| Maria Germany |
| John USA      |

---

# 4️⃣ UPPER() and LOWER() – Change Letter Case

## UPPER()

Converts all characters to uppercase.

```sql
SELECT UPPER(first_name) AS up_name
FROM customers;
```

## LOWER()

Converts all characters to lowercase.

```sql
SELECT LOWER(first_name) AS low_name
FROM customers;
```

---

# 5️⃣ TRIM() – Remove Leading & Trailing Spaces

## Purpose

Removes:

- Leading spaces (beginning)
- Trailing spaces (end)

Example values:

| Value    |
| -------- |
| 'John'   |
| ' John'  |
| 'John '  |
| ' John ' |

After TRIM:

All become:

```
John
```

---

## Detect Leading/Trailing Spaces

Method 1 (Recommended):

```sql
SELECT *
FROM customers
WHERE first_name <> TRIM(first_name);
```

Method 2 (Using Length):

```sql
WHERE LEN(first_name) <> LEN(TRIM(first_name));
```

---

# 6️⃣ REPLACE() – Replace or Remove Characters

## Purpose

Replaces old value with new value.

### Syntax

```sql
REPLACE(value, old_value, new_value)
```

---

### Example 1: Replace Dash

```sql
SELECT REPLACE('123-456-7890', '-', '/');
```

Output:

```
123/456/7890
```

---

### Example 2: Remove Dash

```sql
SELECT REPLACE('123-456-7890', '-', '');
```

Output:

```
1234567890
```

---

### Example 3: Change File Extension

```sql
SELECT REPLACE('report.txt', '.txt', '.csv');
```

Output:

```
report.csv
```

---

# 7️⃣ LEN() – Count Characters

## Purpose

Counts number of characters in a value.

```sql
SELECT LEN('Maria');
```

Output:

```
5
```

Works for:

- Text
- Numbers
- Dates

Example:

```sql
SELECT LEN(first_name) AS length_name
FROM customers;
```

---

# 8️⃣ LEFT() – Extract from Beginning

## Purpose

Extracts specific number of characters from the start.

```sql
SELECT LEFT(first_name, 2) AS first_two
FROM customers;
```

If first_name = Maria

Output:

```
Ma
```

---

# 9️⃣ RIGHT() – Extract from End

## Purpose

Extracts characters from end of string.

```sql
SELECT RIGHT(first_name, 2) AS last_two
FROM customers;
```

If first_name = Maria

Output:

```
ia
```

---

# 🔟 SUBSTRING() – Extract from Middle

## Purpose

Extract part of string from specific position.

### Syntax

```sql
SUBSTRING(value, start_position, length)
```

---

### Example 1: Extract Middle

```sql
SELECT SUBSTRING('Maria', 3, 2);
```

Output:

```
ri
```

---

### Example 2: Remove First Character

```sql
SELECT SUBSTRING(first_name, 2, LEN(first_name))
FROM customers;
```

Explanation:

- Start from position 2
- Extract dynamic length

---

### Best Practice (Clean Version)

```sql
SELECT SUBSTRING(TRIM(first_name), 2, LEN(TRIM(first_name)))
FROM customers;
```

This ensures:

- No leading spaces
- Dynamic extraction

---

# 1️⃣1️⃣ Nested String Functions

You can combine multiple functions together.

Example:

```sql
SELECT LEN(LOWER(LEFT(first_name, 2)))
FROM customers;
```

Execution order:

1. LEFT()
2. LOWER()
3. LEN()

Always inside → outside
