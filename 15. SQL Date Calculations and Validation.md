# SQL Date Calculations and Validation — Advanced Notes

---

# 1. Date Calculations Overview

Date calculations allow us to:

- Add time intervals
- Subtract time intervals
- Measure differences between dates
- Perform time-gap analysis
- Validate date quality

Core functions covered:

- DATEADD()
- DATEDIFF()
- ISDATE()

---

# 2. DATEADD() Function

## 2.1 What is DATEADD?

`DATEADD()` allows you to add or subtract a specific time interval to/from a date.

It modifies one specific date part.

---

## 2.2 Syntax

```sql
DATEADD(part, interval, date)
```

Parameters:

- part → year, month, day, hour, etc.
- interval → Number (positive = add, negative = subtract)
- date → Base date

---

## 2.3 Basic Concept Example

Original Date:

```
2025-08-20
```

Add 3 years:

```sql
SELECT DATEADD(year, 3, '2025-08-20');
```

Result:

```
2028-08-20
```

Subtract 3 years:

```sql
SELECT DATEADD(year, -3, '2025-08-20');
```

Result:

```
2022-08-20
```

---

## 2.4 Add Months

```sql
SELECT DATEADD(month, 2, '2025-08-20');
```

Result:

```
2025-10-20
```

Subtract 2 months:

```sql
SELECT DATEADD(month, -2, '2025-08-20');
```

Result:

```
2025-06-20
```

---

## 2.5 Add/Subtract Days

```sql
SELECT DATEADD(day, 5, '2025-08-20');
```

Result:

```
2025-08-25
```

```sql
SELECT DATEADD(day, -5, '2025-08-20');
```

Result:

```
2025-08-15
```

---

## 2.6 Practical Example Using Table

```sql
SELECT
    OrderDate,
    DATEADD(year, 2, OrderDate) AS TwoYearsLater,
    DATEADD(month, 3, OrderDate) AS ThreeMonthsLater,
    DATEADD(day, -10, OrderDate) AS TenDaysBefore
FROM SalesOrders;
```

# 3. DATEDIFF() Function

## 3.1 What is DATEDIFF?

`DATEDIFF()` calculates the difference between two dates.

It returns an integer.

---

## 3.2 Syntax

```sql
DATEDIFF(part, start_date, end_date)
```

Parameters:

- part → year, month, day
- start_date → earlier date
- end_date → later date

---

## 3.3 Concept Example

Order Date:

```
2025-08-20
```

Shipping Date:

```
2026-02-01
```

Years difference:

```sql
SELECT DATEDIFF(year, '2025-08-20', '2026-02-01');
```

Result:

```
1
```

Months difference:

```sql
SELECT DATEDIFF(month, '2025-08-20', '2026-02-01');
```

Result:

```
6
```

Days difference:

```sql
SELECT DATEDIFF(day, '2025-08-20', '2026-02-01');
```

Result:

```
165
```

# 4. Real-World Use Cases

## 4.1 Calculate Employee Age

```sql
SELECT
    EmployeeID,
    BirthDate,
    DATEDIFF(year, BirthDate, GETDATE()) AS Age
FROM Employees;
```

Execution Logic:

- Start date → BirthDate
- End date → Current system date
- Returns number of years

---

## 4.2 Calculate Shipping Duration

```sql
SELECT
    OrderID,
    DATEDIFF(day, OrderDate, ShipDate) AS ShippingDuration
FROM SalesOrders;
```

---

## 4.3 Average Shipping Duration Per Month

```sql
SELECT
    MONTH(OrderDate) AS OrderMonth,
    AVG(DATEDIFF(day, OrderDate, ShipDate)) AS AvgShippingDays
FROM SalesOrders
GROUP BY MONTH(OrderDate);
```

---

## 4.4 Time Gap Analysis (Between Orders)

Using Window Function:

```sql
SELECT
    OrderID,
    OrderDate,
    LAG(OrderDate) OVER (ORDER BY OrderDate) AS PreviousOrderDate,
    DATEDIFF(day,
        LAG(OrderDate) OVER (ORDER BY OrderDate),
        OrderDate) AS DaysBetweenOrders
FROM SalesOrders;
```

This is called time-gap analysis.

# 5. ISDATE() Function

## 5.1 What is ISDATE?

Validates whether a value can be interpreted as a date.

Returns:

- 1 → Valid date
- 0 → Invalid date

---

## 5.2 Syntax

```sql
ISDATE(value)
```

---

## 5.3 Examples

```sql
SELECT ISDATE('2025-08-20');
```

Result:

```
1
```

```sql
SELECT ISDATE('123');
```

Result:

```
0
```

---

## 5.4 Data Cleaning Example

Scenario: String dates with invalid records

```sql
SELECT
    OrderDate,
    CASE
        WHEN ISDATE(OrderDate) = 1
        THEN CAST(OrderDate AS DATE)
        ELSE NULL
    END AS CleanedDate
FROM RawOrders;
```

Invalid dates become NULL instead of causing errors.

---

# 6. Performance Notes

- DATEADD and DATEDIFF are efficient and index-friendly when used carefully.
- Avoid wrapping indexed columns in functions inside WHERE clauses.
- Use range filtering instead of DATEDIFF in WHERE.

Example (Avoid):

```sql
WHERE DATEDIFF(day, OrderDate, GETDATE()) < 30
```

Better:

```sql
WHERE OrderDate >= DATEADD(day, -30, GETDATE())
```
