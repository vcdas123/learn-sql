## 🔹 Single Row Function Subgroups

### 1. String Functions

Used to manipulate text.

Examples:

- UPPER()
- LOWER()
- LEFT()
- RIGHT()
- LEN()
- CONCAT()

---

### 2. Numeric Functions

Used for numbers.

Examples:

- ROUND()
- CEILING()
- FLOOR()
- ABS()

---

### 3. Date & Time Functions

Used to manipulate dates.

Examples:

- GETDATE()
- DATEADD()
- DATEDIFF()

---

### 4. NULL Handling Functions

Used to handle missing values.

Examples:

- ISNULL()
- COALESCE()

---

## 🔹 Multi Row Functions Subgroups

### 1. Basic Aggregate Functions

- SUM()
- AVG()
- COUNT()
- MIN()
- MAX()

---

### 2. Window (Analytical) Functions

Advanced aggregation functions.

Examples:

- ROW_NUMBER()
- RANK()
- DENSE_RANK()
- SUM() OVER()

Used for advanced data analysis.

# 8️⃣ Core Concept Summary

Single Row Function:

One value in → One value out

Multi Row Function:

Multiple values in → One value out

Nested Functions:

Output of one function becomes input of another.

Execution Order:

Inner function executes first.

---

# 9️⃣ Big Picture Understanding

Single row functions prepare and clean the data.
Multi row functions analyze and summarize the data.

Together, they allow you to:

✔ Transform raw data
✔ Build reports
✔ Clean inconsistent values
✔ Generate insights
✔ Perform advanced analytics
