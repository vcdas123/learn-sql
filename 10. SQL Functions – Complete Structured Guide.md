# 1️⃣ What Is an SQL Function?

An SQL function is a **built-in code block** that:

1. Accepts an input value
2. Processes the value
3. Returns an output value

Think of it like a small machine:

Input → Processing → Output

Example:

```sql
SELECT LOWER('MARIA');
```

Output:

```
maria
```

The function LOWER:

- Takes input: 'MARIA'
- Processes it
- Returns: 'maria'

# 3️⃣ Two Main Categories of SQL Functions

SQL functions are divided into two main groups:

---

# 🔹 A. Single Row Functions

- Accept **one value**
- Return **one value**

One input → One output

Example:

```sql
SELECT UPPER('maria');
```

Output:

```
MARIA
```

The function processes only one value at a time.

---

# 🔹 B. Multi Row Functions (Aggregate Functions)

- Accept **multiple rows**
- Return **one value**

Multiple inputs → One output

Example:

```sql
SELECT SUM(salary) FROM employees;
```

If salaries are:

| salary |
| ------ |
| 1000   |
| 2000   |
| 3000   |

Output:

```
6000
```

The function SUM:

- Takes multiple values
- Returns a single aggregated result

---

# 4️⃣ Nested Functions

SQL allows you to use functions inside other functions.

Example:

Step 1: Extract first two characters

```sql
SELECT LEFT('MARIA', 2);
```

Output:

```
MA
```

Step 2: Convert to lowercase

```sql
SELECT LOWER(LEFT('MARIA', 2));
```

Output:

```
ma
```

Step 3: Get length of result

```sql
SELECT LEN(LOWER(LEFT('MARIA', 2)));
```

Output:

```
2
```

---

# 5️⃣ Execution Order of Nested Functions

SQL executes functions from inside to outside.

For this query:

```sql
SELECT LEN(LOWER(LEFT('MARIA', 2)));
```

Execution order:

1. LEFT('MARIA', 2)
2. LOWER(result)
3. LEN(result)

Always inside → outside

# 6️⃣ Function Categories in Detail

## 🔹 Single Row Function Subgroups

### 1. String Functions

Used to manipulate text.

Examples:

- UPPER()
- LOWER()
- LEFT()
- RIGHT()
- LEN()
- CONCAT()

---

### 2. Numeric Functions

Used for numbers.

Examples:

- ROUND()
- CEILING()
- FLOOR()
- ABS()

---

### 3. Date & Time Functions

Used to manipulate dates.

Examples:

- GETDATE()
- DATEADD()
- DATEDIFF()

---

### 4. NULL Handling Functions

Used to handle missing values.

Examples:

- ISNULL()
- COALESCE()

---

## 🔹 Multi Row Functions Subgroups

### 1. Basic Aggregate Functions

- SUM()
- AVG()
- COUNT()
- MIN()
- MAX()

---

### 2. Window (Analytical) Functions

Advanced aggregation functions.

Examples:

- ROW_NUMBER()
- RANK()
- DENSE_RANK()
- SUM() OVER()

Used for advanced data analysis.

# 8️⃣ Core Concept Summary

Single Row Function:

One value in → One value out

Multi Row Function:

Multiple values in → One value out

Nested Functions:

Output of one function becomes input of another.

Execution Order:

Inner function executes first.
